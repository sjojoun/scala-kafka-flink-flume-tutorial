## 🚀 Quick Start | 快速开始

### Prerequisites | 前置要求

- JDK 11 or 17
- Scala 2.13+
- 8GB+ RAM
- 50GB+ disk space

### What You'll Learn | 你将学到

✅ **Flume** - Distributed log collection system  
✅ **Kafka** - High-throughput message queue  
✅ **Flink** - Stream processing engine  
✅ **Scala** - Functional programming language
